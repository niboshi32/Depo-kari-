独自システム
アームチェンジシステム

武装の交換や修理ができる（相棒が遠隔で治す）

敵の機械を倒すと部品とバッテリーの補充ができる

アクション：攻撃ｊ、必殺技ｋ、特殊アクションｌ

ミリタリー：軍事用、高性能反応速度と精密動作に優れる
インダストリー：工業用、出力、耐久に優れる
インフォームド：情報特化、情報収集とハックに優れる
（バイオミメティクス）：動物をまねた四肢、鳥の翼や猫の足など。
